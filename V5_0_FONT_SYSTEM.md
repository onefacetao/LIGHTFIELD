
V5.0 LIGHTFIELD FONT SYSTEM REBUILD

1. BRAND TYPOGRAPHY PRINCIPLE
- Typography is not decoration, it is identity structure
- Lightfield wordmark becomes a controlled optical system
- Chinese "光域" is no longer a font selection, but a designed logotype

2. ENGLISH LOGOTYPE (LIGHTFIELD)
- Kerning rebalanced for optical symmetry
- Increased negative space between L / I / G segments
- Reduced mechanical uniformity, introduced micro-optical rhythm
- Baseline alignment locked to grid system

3. CHINESE LOGOTYPE (光域)
- Custom logotype treatment required (NOT system font)
- Stroke contrast unified with English system
- Structural alignment optimized for square composition
- Designed to coexist with LIGHTFIELD, not subordinate

4. LOGO SYSTEM HIERARCHY
PRIMARY: LIGHTFIELD (EN)
SECONDARY: 光域 (CN)
SUBMARK: L monogram icon

5. APPLICATION RULES
- Do not use default system fonts for brand name
- Minimum spacing rule enforced around logotype
- Light source / negative space concept must be consistent

6. VISUAL TONE SHIFT
- From "design portfolio typography"
- To "brand identity typography system"

NOTE:
This is a SYSTEM DESIGN LAYER, not visual layout change.
