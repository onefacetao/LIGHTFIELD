# Lightfield Portfolio 1.0 Final Release

本版本为 Lightfield Official Website / Portfolio 的正式发布候选版。

## 本次重点
- 统一首页品牌语言：看见未被看见 / 价值因被感知而存在 / Discover · Connect · Reveal
- 精简案例内部重复图片
- 删除保利云上不适合展示的时间线/目录类图片
- 调整原乡郡与南岸公馆的封面和图片权重
- 统一中英文标题层级
- 为案例详情页增加 Project Statement
- 保持 Lightfield 黑色空间感与光域品牌气质

## 推荐用途
- 对外官网
- 作品集投递
- 客户沟通展示
- 未来新增案例的基础工程

## 后续维护原则
不再推翻整体结构，只允许：
1. 新增案例
2. 替换图片
3. 微调文案与间距
4. 小幅优化动效


## v1.1 Production Fix Notes

This release focuses on the issues identified after deployment:
1. Incorrect logo mark removed from hero.
2. Signature text refined toward a handwritten brand signature.
3. Contact headline softened and aligned with Lightfield philosophy.
4. Case detail image lightbox added for professional portfolio viewing.
