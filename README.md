# Lightfield Portfolio 1.0

Lightfield Official Website / Portfolio.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Deploy

Recommended: Vercel.

Upload / push the project root containing:

- app/
- data/
- public/
- package.json
- next.config.js
- README.md

## Notes

This is the official Lightfield Portfolio 1.0 Final Release package.
Do not restructure the site unless starting a future major version.
